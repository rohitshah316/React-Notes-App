# 📝 React Notes App

A simple and efficient **React Notes App** that allows users to quickly create, edit, and manage their notes with a clean and user-friendly interface.

This app is designed to demonstrate core React concepts like **state management, routing, and persistent storage**, while maintaining a modern UI.

## 🛠️ Features

- ✍️ Create new notes instantly  
- 📝 Edit notes in real-time  
- ❌ Delete notes easily  
- 🔗 Dynamic routing for individual note pages  
- 💾 Persistent storage using **localStorage**  
- 📱 Fully **responsive design**  
- 🎨 Clean UI built with **Tailwind CSS**

---

## ⚙️ Tech Stack

- **React JS**
- **Vite**
- **Tailwind CSS**
- **React Router**
- **Context API**
- **LocalStorage**
